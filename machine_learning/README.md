## GBDT等树模型为啥不适合处理高纬度稀疏特征

树模型可以直接处理分类特征，不需要做onehot。onehot后的特征是高度稀疏的，如果刚好某一个值的结果一致可能导致过拟合。
树模型的正则惩罚项是针对深度和叶子节点数，对这种情况下的惩罚项极小。
